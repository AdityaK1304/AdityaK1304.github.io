# Aditya Kante - Data Analyst Portfolio

My personal portfolio website showcasing data analytics projects, skills, and experience.

## Live Website
Visit: [https://adityak1304.github.io](https://adityak1304.github.io)

## Features
- Responsive design that works on all devices
- Showcase of data analytics projects
- Skills section with proficiency levels
- Education and certifications
- Contact form

## How to Update

### 1. Add Your Photo
- Rename your photo to `profile.jpg`
- Upload it to the repository in the same folder as `index.html`

### 2. Add Your Resume
- Rename your resume to `resume.pdf`
- Upload it to the repository

### 3. Update Information
Edit the `index.html` file directly to:
- Update contact information
- Add new projects
- Update skills
- Modify any text content
